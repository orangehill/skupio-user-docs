# Skupio documentation

Skupio is a privacy-first lead pipeline for cold email: LinkedIn lead capture through your own browser, AI-authored email sequences, and delivery through Instantly.

Three places to start, depending on where you are.

## New here? Build your first campaign

If you have not used Skupio yet, the fastest path to a working campaign is one tutorial:

→ **[Your first scrape and campaign](/tutorials/your-first-scrape)** — install the Chrome extension, pick an audience, scrape 50 leads, let the AI write a 5-step sequence, push it to Instantly. About 30 minutes.
